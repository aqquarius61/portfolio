# Portfolio Website

HTML, CSS, and basic JavaScript로 만든 간단한 반응형 포트폴리오 웹사이트입니다.

## Pages

- `index.html`: 자기소개 페이지
- `activities.html`: 활동 목록 페이지

## Files

- `style.css`: 전체 디자인과 반응형 레이아웃
- `script.js`: 모바일 메뉴와 활동 필터 기능

## How to View

1. 이 저장소를 다운로드하거나 클론합니다.
2. `index.html` 파일을 브라우저에서 엽니다.
3. 상단 메뉴에서 자기소개 페이지와 활동 목록 페이지를 이동할 수 있습니다.

## Features

- Modern material-inspired design
- Responsive layout for mobile and desktop
- Two-page structure
- Activity category filtering with JavaScript

## Customize

`index.html`과 `activities.html` 안의 이름, 소개 문장, 활동 내용을 본인 정보로 수정하면 됩니다.
